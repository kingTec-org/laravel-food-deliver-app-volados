## Get Started

Composer install
```sh
$ composer install
```

Laravel key generate
```sh
$ php artisan key:generate
```

Laravel storage link
```sh
$ php artisan storage:link
```

Laravel database migrate and seed
```sh
$ php artisan migrate --seed
```

Run serve
```sh
$ php artisan serve
```
